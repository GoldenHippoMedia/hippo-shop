Two ways to reach a variant: standardList is the ordered array you iterate with <template data-each>, while standardByQuantity.<qty> is a direct keyed lookup for the cases where you know exactly which package you want to bind.

Every section root carries a data-if guard — wrap any block in data-if="path.to.data" and it cleanly hides when the data isn't there, no JavaScript required. Empty arrays are truthy as objects, so use list.0 to mean "list has at least one entry".
