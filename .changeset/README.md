# Changesets

Hi! This folder has been automatically generated by `@changesets/cli`, a tool to manage versioning and changelogs.

Common commands:

```
pnpm changeset           # add a new changeset entry
pnpm version-packages    # consume changesets and bump versions
pnpm release             # build + publish
```

See the [changesets docs](https://github.com/changesets/changesets) for full reference.
